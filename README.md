# AI-Dungeon-Game
Dungeon Game led by gemini and using tkinter.

**The model used is gemini-1.5-flash. Get API Key from aistudio.google.com. Run the following code in bash:**
```
$ export API_KEY=YourApiKeyThatYouGot
$ python main.py
```
